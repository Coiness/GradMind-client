#### 任务 4: 梯度计算（1.5小时）
**文件**: `src/config/algorithms/analyticalOptimization/gradientComputation.ts`

**输入**:
- `objective`: 目标函数（字符串或函数对象）
- `point`: 计算梯度的点（向量）
- `epsilon`: 步长（默认 1e-5）

**输出**:
```typescript
{
  gradient: number[],      // 梯度向量
  point: number[],         // 输入点
  norm: number            // 梯度范数
}
```

**算法**:
```
对于每个维度 i:
  ∂f/∂x_i ≈ [f(x + εe_i) - f(x - εe_i)] / (2ε)
```

**实现要点**:
- 解析字符串函数（使用 mathjs.parse）
- 数值微分
- 处理多维输入

#### 任务 5: 最小二乘法（2小时）
**文件**: `src/config/algorithms/analyticalOptimization/leastSquares.ts`

**输入**:
- `x`: 自变量数据（矩阵或向量）
- `y`: 因变量数据（向量）

**输出**:
```typescript
{
  coefficients: number[],  // 回归系数 β
  predictions: number[],   // 预测值 ŷ
  residuals: number[],     // 残差 y - ŷ
  rSquared: number,        // R² 决定系数
  mse: number             // 均方误差
}
```

**算法**:
```
β = (X^T X)^(-1) X^T y
ŷ = Xβ
R² = 1 - SS_res / SS_tot
```

**实现要点**:
- 添加截距项（如果需要）
- 矩阵求逆（使用 mathjs.inv）
- 计算统计指标

#### 任务 7: PCA（2.5小时）
**文件**: `src/config/algorithms/dataReduction/pca.ts`

**输入**:
- `dataset`: 输入数据（矩阵）
- `nComponents`: 主成分数量（默认 2）

**输出**:
```typescript
{
  components: number[][],        // 主成分（特征向量）
  variance: number[],           // 方差
  explainedVariance: number[],  // 解释方差比
  transformed: number[][]       // 转换后的数据
}
```

**算法**:
```
1. 中心化数据: X_centered = X - mean(X)
2. 计算协方差矩阵: Cov = (X_centered^T × X_centered) / (n-1)
3. 特征值分解: Cov = VΛV^T
4. 选择前 k 个特征向量
5. 转换数据: Y = X_centered × V_k
```

**实现要点**:
- 数据中心化
- 协方差矩阵计算
- 特征值分解（使用 mathjs.eigs）
- 按特征值排序

#### 任务 8: Hessian矩阵（2.5小时）
**文件**: `src/config/algorithms/analyticalOptimization/hessianMatrix.ts`

**输入**:
- `objective`: 目标函数
- `point`: 计算点
- `epsilon`: 步长

**输出**:
```typescript
{
  hessian: number[][],    // Hessian矩阵
  eigenvalues: number[],  // 特征值
  isPositiveDefinite: boolean  // 是否正定
}
```

**算法**:
```
H_ij = ∂²f/∂x_i∂x_j
使用数值方法：
H_ij ≈ [f(x+εe_i+εe_j) - f(x+εe_i-εe_j) - f(x-εe_i+εe_j) + f(x-εe_i-εe_j)] / (4ε²)
```

#### 任务 9: 梯度下降（3小时）
**文件**: `src/config/algorithms/numericalOptimization/gradientDescent.ts`

**输入**:
- `objective`: 目标函数
- `initialPoint`: 初始点
- `learningRate`: 学习率（默认 0.01）
- `maxIterations`: 最大迭代次数（默认 100）
- `tolerance`: 收敛容差（默认 1e-6）

**输出**:
```typescript
{
  optimalPoint: number[],     // 最优点
  optimalValue: number,       // 最优值
  iterations: number,         // 实际迭代次数
  history: {                  // 历史记录
    points: number[][],
    values: number[],
    gradients: number[][]
  },
  converged: boolean         // 是否收敛
}
```

**算法**:
```
重复直到收敛:
  1. 计算梯度: g = ∇f(x)
  2. 更新: x = x - α·g
  3. 检查收敛: ||g|| < tolerance
```

**文件**: `src/config/algorithms/analyticalOptimization/lagrangeMultiplier.ts`

**输入**:
- `objective`: 目标函数 f(x)

**输出**:
```typescript

```

#### 任务 12: 高级优化器（4小时）
**文件**: `src/config/algorithms/numericalOptimization/advancedOptimizer.ts`

**输入**:
- `objective`: 目标函数
- `initialPoint`: 初始点
- `optimizer`: 优化器类型（'adam' 或 'rmsprop'）
- `learningRate`: 学习率
- `beta1`, `beta2`: Adam 参数
- `maxIterations`: 最大迭代次数

**输出**:
```typescript
{
  optimalPoint: number[],
  optimalValue: number,
  history: {
    points: number[][],
    values: number[]
  },
  converged: boolean
}
```

**Adam 算法**:
```
m_t = β1·m_{t-1} + (1-β1)·g_t
v_t = β2·v_{t-1} + (1-β2)·g_t²
m̂_t = m_t / (1-β1^t)
v̂_t = v_t / (1-β2^t)
x_t = x_{t-1} - α·m̂_t / (√v̂_t + ε)
```

#### 任务 15: 神经网络（7-9小时）
**文件**: `src/config/algorithms/numericalOptimization/neuralNetwork.ts`

**输入**:
- `x`: 训练数据
- `y`: 标签
- `hiddenLayers`: 隐藏层配置 [64, 32]
- `activation`: 激活函数（'sigmoid', 'relu', 'tanh'）
- `epochs`: 训练轮数
- `learningRate`: 学习率

**输出**:
```typescript
{
  model: {                 // 训练好的模型
    weights: number[][][],
    biases: number[][]
  },
  history: {
    loss: number[],
    accuracy: number[]
  },
  predictions: number[]   // 对训练数据的预测
}
```

**算法**:
```
前向传播:
  z^[l] = W^[l]·a^[l-1] + b^[l]
  a^[l] = activation(z^[l])

反向传播:
  δ^[L] = (a^[L] - y) ⊙ activation'(z^[L])
  δ^[l] = (W^[l+1])^T·δ^[l+1] ⊙ activation'(z^[l])

参数更新:
  W^[l] = W^[l] - α·δ^[l]·(a^[l-1])^T
  b^[l] = b^[l] - α·δ^[l]
```

**实现要点**:
- 矩阵初始化（Xavier/He）
- 激活函数及其导数
- 前向传播
- 反向传播
- 损失函数（MSE/交叉熵）
- 训练循环

## 📝 注意事项

### 通用实现原则：
1. **数据提取**: 始终从 inputs 中正确提取数据
   ```typescript
   const datasetInput = inputs.dataset;
   const data = datasetInput?.data || datasetInput;
   ```

2. **参数使用**: 确保参数实际影响计算
   ```typescript
   const learningRate = Number(params.learningRate) || 0.01;
   ```

3. **错误处理**: 捕获和报告错误
   ```typescript
   try {
     // 计算逻辑
   } catch (error) {
     throw new Error(`算法执行失败: ${error.message}`);
   }
   ```

4. **数据验证**: 检查输入数据的有效性
   ```typescript
   if (!data || data.length === 0) {
     throw new Error('输入数据为空');
   }
   ```

5. **返回格式**: 保持与原 mock 实现相同的输出结构

### 数学库使用：
- 矩阵运算: `math.multiply()`, `math.transpose()`, `math.inv()`
- 统计函数: `math.mean()`, `math.std()`, `math.variance()`
- 线性代数: `math.eigs()`, `math.det()`, `math.norm()`
- 函数解析: `math.parse()`, `math.compile()`
