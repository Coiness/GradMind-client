## 测试用例

### 测试 1: 数据降维 - PCA

**目标**: 验证 PCA 算法能正确降维

**步骤**:
1. 点击"添加数据集"按钮
2. 选择"手动输入"标签页
3. 输入以下数据（Iris 数据集样本）:
   ```
   5.1,3.5,1.4,0.2
   4.9,3.0,1.4,0.2
   4.7,3.2,1.3,0.2
   4.6,3.1,1.5,0.2
   5.0,3.6,1.4,0.2
   ```
4. 点击"添加到画布"
5. 从左侧算法库拖拽"PCA（主成分分析）"到画布
6. 连接数据集节点的输出到 PCA 的"输入数据集"端口
7. 在右侧面板设置参数：
   - 成分数量: 2
   - 数据中心化: 是
8. 点击"执行"按钮

**预期结果**:
- ✅ 执行成功，无错误
- ✅ 结果标签页显示：
  - `components`: 2×4 矩阵（主成分）
  - `variance`: 2 个数值（方差）
  - `explainedVariance`: 2 个数值，总和 < 1
  - `transformed`: 5×2 矩阵（转换后的数据）
- ✅ 第一个主成分的解释方差比 > 0.7

import { algorithmLibrary } from '@/config/algorithms';

  it('PCA 应该正确降维', async () => {
    const workflow = {
      nodes: [
        { id: '1', type: 'dataset', data: { datasetData: { data: [[1,2],[3,4]] } } },
        { id: '2', type: 'algorithm', data: { algorithmKey: 'pca', parameters: { nComponents: 2 } } }
      ],
      edges: [
        { id: 'e1', source: '1', target: '2', sourceHandle: 'dataset', targetHandle: 'dataset' }
      ]
    };

    const results = await executeWorkflowEngine(workflow, algorithmLibrary);

    expect(results['2']).toBeDefined();
    expect(results['2'].components).toBeDefined();
    expect(results['2'].variance).toBeDefined();
