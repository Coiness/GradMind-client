# 📋 项目内容总结 - 前端开发需求

## 🎯 项目核心概述

**项目名称**：知几 - AI数学原理交互式学习工具箱  
**项目目标**：开发一个Web平台的交互式AI数学学习工具箱，帮助学生直观理解AI数学原理

## 🚀 需要实现的前端功能模块

### **1. 可视化交互模块**

```
功能：通过交互式界面展示AI数学算法的动态过程
具体需求：
- 参数调节组件（滑块、输入框、下拉菜单）
- 实时可视化渲染（2D/3D图表）
- 即时反馈机制（参数调整立即看到效果）
- 多视角切换功能
技术栈：React + ECharts + Three.js + Canvas
```

### **2. 公式-算法对译模块**

```
功能：建立数学公式与代码的直观对应关系
具体需求：
- LaTeX数学公式渲染
- 代码语法高亮显示
- 双向悬停高亮联动
- 响应式双栏布局
技术栈：MathJax + Prism.js + React
```

### **3. 算法流编排模块**

```
功能：拖拽式算法组合和流程构建
具体需求：
- 可拖拽的算法模块库
- 可视化连接画布
- 模块连接验证
- 运行状态显示
技术栈：react-beautiful-dnd + 自定义连接线
```
